## 簡述
以vue-element-admin為基本範例，結合ELEMENT-UI和VUE-CLI，開發出的後台管理網站，以routers去控制後台的主要架構，並依照每個帳戶的角色動態生成router


## Build Setup

```bash


# 安装依赖
yarn install


# 启动服务
yarn serve
```

## 發布

```bash
# 构建测试环境
yarn build:stage

# 构建生产环境
yarn build:prod```


# 代码格式检查
yarn lint

# 自动修复
yarn lint -- --fix
```
